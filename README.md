# Investment Research Share Site

Generated by `scripts/build_share_site.ps1`.

Included public content:

- Podcast analysis JSON
- AI full reports
- Taiwan watchlists and XQ import lists
- Stock analysis notes
- Sector analysis notes
- Taiwan stock universe

Intentionally excluded:

- Personal portfolio
- Bank balance and settlement cash
- Realized trade records
- Local caches, logs, transcripts, and pending analysis files

Deploy the whole `share` folder to GitHub Pages, Cloudflare Pages, Netlify, or another static hosting service. The entry file is `index.html`.
