## 整集摘要

本集由 Ergotron 螢幕支架贊助，主持人以每天盯盤工作至少 8 小時為例，認為交易環境與身體姿勢是基本盤，螢幕支架可改善脖子、肩膀與雙螢幕空間配置；優惠碼 GOOAYE 全管螢幕支架 9 折。（廣告，非投資）

生活段先談紅酒探險。主持人用「金錢戰術」快速建立紅酒識別度，透過喝五大酒莊、Cult Vineyard、不同年份與地塊來校準味覺，類比當年喝清酒後才找到自己喜歡的新政三千多塊品項；他強調最好的酒不一定是 price tag 或市場封神，而是找到自己喜歡的味道。Costco 段則分享在南崁店買到木桐、拉菲等五大酒莊中的三個，價格甚至比自己酒商便宜八千，但酒可能偏年輕不能太早開。（生活觀察）

減肥段分享從 Ozempic 換到 Mounjaro。醫師開 5 毫克，他先照建議從 2.5 開始，結果幾乎沒有感覺，食慾仍要靠意志力控制，唯一明顯副作用是晚上口渴；體重仍微幅下降並接近目標，下次可能拉高劑量。（生活觀察）

市場段先談 CPU。Intel 數字不錯且後續要漲價，CPU 類股全面噴出；主持人說先前是從 Agentic AI 需要 CPU 的產業推理與技術面端倪出發，不是事後收割。市場開始有人把 server rack 中 CPU 與 GPU/ASIC 配置喊成 1:1，但主持人認為目前主流更可能是 2G1C 或 4G1C，短期看到 Intel、AMD 噴出後追小作文有受傷風險；長期 CPU 趨勢可能剛啟動但要邊走邊看。（偏多但提醒短線過熱）

TPU 段提到 8T/8I 正式揭曉，對應 Zebrafish 與 Soundfish，由聯發科與 Broadcom 設計；主持人原先誤以為 Training 是 Broadcom、Inference 是 MTK，後來得知 Training 是 MediaTek。產業朋友推估 Training 量可能略多，但也可能趨近 1:1，因此 MTK 後續有望受惠；聯發科短期已有三根漲停，主持人非常感恩。（偏多）

本集核心是台積電投信持股上限從 10% 拉到 25%。主持人把這視為替台積電設計的法規，等於解開封印；過去 ETF 為了拉高台積電 exposure 會去買其他 ETF，現在可直接提高持股。主持人大幅減碼其他持股，禮拜五比較大部位買台積電，估算若 ETF 要買到 25%，大約要買十幾萬張、十萬多張出頭；即使不抓滿，只要 3 萬張或 5 萬張也足以形成資金行情，而禮拜五投信盤後只買 1000 多張，代表錢可能尚未完全進場。台積電明年共識 EPS 可算到 140 元，用 20 倍是 2800，前幾集才推過台股上 4 萬點；若資金與政策催化讓市場願意給 25 倍、30 倍或看 2028/2029，心中 TP 還可上修。（大力看好，但前提是外資不要倒貨）

主持人同時說明反面：若投信與經理人要把台積電從 10% 拉到 15%、18% 或更高，錢一定要從原本持股來，因此可能造成中小型股被賣、垃圾盤擴大。他把一堆中小丟掉去買台積電，認為禮拜四、禮拜五 Smart Money 也可能在做同樣事。若台積電真的被大量資金推上去，會引發 FOMO 逼迫更多資金從其他標的轉來；但等資金重平衡完成，被賣爛但基本面撐住的中小型股，反而可能成為下一輪類股輪動機會。整體結論是下週開始要注意政策驅動資金行情與波動，波動就是交易機會。（事件驅動，積極但有風險）

## 研究任務

### Step 1: 驗證台積電政策買盤是否啟動
**目的：** 本集最重要假設是投信持股上限放寬後，台積電會迎來制度性買盤。

**需要資料：** 台積電每日投信買賣超、台積電每日外資買賣超、ETF 持股比重、台積電成交量與股價位置

**使用方式：** 若投信買超逐步放大且外資沒有大幅倒貨，政策資金行情可信度提高；若投信不買或外資賣壓更大，劇本降權。

### Step 2: 建立賣中小買台積電的受影響觀察池
**目的：** 主持人認為提高台積電比重的資金會從原本持股流出，短期可能造成中小垃圾盤，之後可能有補部位行情。

**需要資料：** 近期投信買賣超排行、櫃買指數相對加權指數、中小型強勢股回檔清單、月營收與財報、成交量與籌碼

**使用方式：** 篩出只是被資金調節、基本面未壞的標的作為觀察池；若基本面也轉弱則排除。

### Step 3: 檢查台積電估值是否被市場充分預期
**目的：** 主持人用 EPS 140、20 倍 2800 作為估值框架，並思考 25 倍、30 倍或看 2028/2029 的可能。

**需要資料：** 券商目標價區間、市場 EPS 共識、AI peer 本益比、台積電法說與財測、股價相對估值位置

**使用方式：** 若券商 TP 普遍高於主持人心中估值或市場敘事過度一致，代表預期可能已充分反映；若估值仍低於 AI peer 且資金未完全進場，主線維持。

### Step 4: 建立 server CPU supply chain 候選池並排除過熱敘事
**目的：** CPU 類股長線偏多，但主持人提醒 1:1 配置小作文過度外推，不能只追題材。

**需要資料：** 公司年報與產品線、法說是否提及 server CPU、月營收、毛利率、CPU 相關報價或交期、股價短線漲幅

**使用方式：** 優先保留 server side 曝險清楚且營收可驗證者；只靠 1:1 配置故事、沒有營收支撐的標的降權。

### Step 5: 追蹤 TPU 8T/8I 設計分工與聯發科驗證資料
**目的：** 主持人修正認知後認為 Training 是 MediaTek，這可能影響市場對聯發科 AI server 曝險的估算。

**需要資料：** 聯發科法說、Broadcom 法說、TPU 8T/8I 相關公開資訊、月營收、ASIC 產品線揭露

**使用方式：** 若法說與營收支持 Training 量較大或接近 1:1，MTK/TPU 主題維持；若量比或設計歸屬被推翻，需重新檢查假設。

## 決策框架

**使用方式**
- 先把台積電政策行情當事件驅動研究，不要只用傳統估值看；核心驗證是投信買超是否真的放大，以及外資是否停止倒貨。
- 台積電越強，中小型短期越可能被抽資金；但被賣不等於基本面壞，應分開檢查籌碼調節與營收惡化。
- 台積電心中 TP 可隨資金與本益比假設上修，但若市場與券商目標價已喊得更高，就要警覺預期過滿。
- CPU 主題長期可偏多，但不能把行情好時冒出的 1:1 配置文章直接當基本面，要回到 server rack 實際配置與營收驗證。
- TPU/MTK 主題的關鍵不是名字好不好聽，而是 Training/Inference 量比與設計歸屬能否在法說、月營收與供應鏈資料中被確認。
- 普通投資人的核心配置與主動交易要分開；主持人仍提醒 0050、006208 這類指數工具更適合多數人作為主軸。
- 股品與交易圈信用會影響長期機會，不要聚眾拉人進場後自己砸盤，這會讓未來好機會遠離你。
